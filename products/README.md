# JEE Product Suite

This folder contains a sellable digital product stack for JEE aspirants built around execution, accountability, and shortcuts rather than generic theory.

## Products

1. `01-jee-rank-accelerator-blueprint.md`
   The flagship operating system for a student trying to improve rank in limited time.
2. `02-study-systems-pack/`
   Ready-to-use planning and review templates in CSV format for Sheets, Excel, or Notion import.
3. `03-mistake-book-templates/`
   Subject-specific mistake logs designed to turn repeated errors into measurable improvement.
4. `04-ai-student-toolkit/50-chatgpt-prompts-for-jee.md`
   A differentiated prompt pack that helps students use AI for revision, testing, and doubt-solving.

## Suggested Offer Stack

- Front-end offer: `JEE Rank Accelerator Blueprint`

Help a serious JEE student stop guessing what to study next, recover from backlog, analyze mistakes faster, and build a repeatable system for score improvement.
