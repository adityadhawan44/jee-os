# Study Systems Pack

Every file in this folder is built to be imported into Google Sheets, Excel, or Notion.

## Files

- `revision-tracker.csv`
  Tracks spaced revision chapter by chapter.
- `rank-improvement-tracker.csv`
  Shows whether score growth is coming from accuracy, attempts, or stronger subjects.
- `daily-study-dashboard.csv`
  Forces daily execution review instead of vague studying.
- `backlog-elimination-planner.csv`
  Turns backlog into a finite recovery project.
- `mock-test-review-sheet.csv`
  Converts every test into improvement data.
- `habit-tracker.csv`
  Tracks the boring habits that support marks.

## Best Way To Sell It

Position it as:

`Plug-and-play student systems. Open the sheet, duplicate it, and start using it in 10 minutes.`

- CSV for universal use
