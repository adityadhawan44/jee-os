# Mistake Book Templates

This product works because students usually repeat mistakes without documenting them clearly.

The job of this pack is to make mistakes searchable, reviewable, and correctable.

## Files

- `physics-error-log.csv`
  Use for derivation misses, sign errors, unit misses, and wrong model selection.
- `chemistry-formula-mistake-tracker.csv`
  Use for formulas, reactions, named exceptions, and confusion-heavy memorization.
- `maths-pattern-journal.csv`
  Use for question types that keep reappearing with different wording.
- `questions-i-always-mess-up.csv`
  Use for recurring failure patterns across subjects.

## Core Selling Angle

Most students revise chapters.
Very few revise their own mistakes.

This pack helps them do exactly that.
